# DDNS - E-Studie

Triggers a DDNS update via a cPanel webcall on `dragic.com`.

## Command

```bash
curl -s -L "https://dragic.com/cpanelwebcall/"
```

## Options

| Option | Description |
| ------ | ----------- |
| `-s` | Silent mode — suppresses progress output |
| `-L` | Follow redirects |

## Usage

```bash
bash "DDNS - E-Studie.sh"
```
